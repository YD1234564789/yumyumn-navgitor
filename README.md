## 執行方式

1.請先確認有安裝 node.js 與 npm

2.將專案 clone 到本地

    git clone https://github.com/sigox/3S2A5.git

3.開啟兩個Terminal，分別使用 cd client 及 cd server

    cd client 
    cd server

4.兩邊皆安裝 npm 套件

    npm install

5.server啟動專案

    node app.js
    
6.client啟動專案

    npm start
    
5.若欲暫停使用

### `ctrl + c`
